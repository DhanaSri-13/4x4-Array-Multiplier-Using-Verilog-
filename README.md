# 4x4-Array-Multiplier-Using-Verilog-

## Description
Designed a combinational 4-bit array multiplier in Verilog HDL using structured partial product generation and carry propagation. Implemented using only Half and Full Adders for parallel binary multiplication. Verified correctness with fixed and random test cases through simulation waveform analysis.
Designed and simulated using Vivado.

## Tools Used
- Vivado
- Verilog HDL

## Files Included
- Design source file (array multiplier)
- Half Adder and Full Adder modules
- Testbench file


## How to Run
1. Open Vivado  
2. Create a new project and add design source files  
3. Add the testbench file  
4. Set testbench as top module  
5. Run Behavioral Simulation  
6. Run Synthesis (optional)
